Quick start:

1. npm install
2. npm run dev

This is a Vite + React + TypeScript app.

UI: This project uses shadcn/ui. To manage components, use the `shadcn` CLI:

- Initialize: `npx shadcn@latest init` (choose a base color)
- Add components: `npx shadcn@latest add <component>` (e.g. `button`)

Tailwind: `tailwind.config.cjs` and `src/index.css` are configured for shadcn.
